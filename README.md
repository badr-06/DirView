# DirView - Qt File System Viewer

## О проекте

**DirView** — это графическое приложение для просмотра файловой системы, написанное на C++ с использованием фреймворка Qt. Программа предоставляет удобный интерфейс для навигации по файлам и папкам, с поддержкой фильтрации, сортировки и расчёта размера папок.

Проект разработан на основе официального примера `itemviews/dirview` из Qt, но значительно расширен и доработан.

---

## Возможности

- **Просмотр файловой системы** — отображение файлов и папок в виде дерева
- **Скрытые файлы** — поддержка отображения скрытых файлов и папок (начинающихся с `.`)
- **Фильтрация** — поиск файлов и папок по имени в реальном времени
- **Размер папок** — вычисление и отображение размера папок по запросу
- **Обновление размера** — кнопка "Обновить" для вычисления размера папки
- **Сортировка** — сортировка файлов по имени, размеру, типу и дате
- **Папки сверху** — папки отображаются перед файлами
- **Скрытые файлы снизу** — скрытые файлы и папки отображаются в конце списка

## Сборка и установка

### Требования

- **Qt 5.15** или выше
- **CMake 3.10** или выше
- **C++17** или выше

### Ubuntu / Debian

#### сборка с помощью cmake

```bash
# 1. Клонировать репозиторий
git clone https://github.com/yourusername/DirView.git
cd DirView

# 2. Установка зависимостей
sudo apt install -y cmake qtbase5-dev qtbase5-dev-tools build-essential

# 3. Собрать проект
make
```

#### сборка с помощью qmake
```bash
# 1. Собрать проект
mkdir -p build
cd build
qmake ..
make
```

#### собрать 
```bash
# 1. Установка зависимостей
sudo apt install -y debhelper devscripts

# 2. Собрать пакет
dpkg-buildpackage -us -uc

```
